• Intent
    ►
    • Activities
      ►
      • UI component of Android apps
      • Interact with users
      • No main() function like typical Java apps
    • Linking Activities
      ►
      • Use intents to navigate between activities
    • Intent Use Cases
      ►
      • Start an activity
      • Start a service
      • Deliver a broadcast
    • Types of Intents
      ►
      • Explicit Intents
        ►
        • Specify component by class name
        • Used within the same app
        • Example: startActivity to launch new screen
      • Implicit Intents
        ►
        • No specific component specified
        • Action declared, handled by any capable app
        • Example: view map using Google Maps
    • Building an Intent
      ►
      • Component Name
      • Action (e.g., ACTION_VIEW, ACTION_SEND)
      • Data (e.g., URI)
      • Extras
        ►
        • Key-value pairs (putExtra)
        • Bundles for multiple values (putExtras)
