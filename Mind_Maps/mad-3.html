• Android App Creating and Activities
    ►
    • Android Emulator
      ►
      • Android Virtual Device (AVD)
      • Simulates real device features
      • Useful for app testing
      • Predefined configurations
      • Alternatives: Genymotion, Bluestacks
    • Creating an Android App
      ►
      • Use Android Studio
      • Set project name and location
      • Select API level
      • Choose Activity type (e.g., Empty)
      • Enter activity info & finish
    • Deploying the App
      ►
      • Use real Android device or AVD
      • Install USB drivers
      • Enable developer mode
      • Use Run button to deploy
    • AndroidManifest.xml
      ►
      • Describes app components
      • Lists activities and permissions
      • Specifies API level & hardware features
      • Links to external libraries
    • Activities in Android
      ►
      • Represents app UI screen
      • Main entry point: MainActivity
      • No main() function
    • Activity Lifecycle
      ►
      • onCreate()
      • onStart()
      • onResume()
      • onPause()
      • onStop()
      • onRestart()
      • onDestroy()
    • Configuring Manifest
      ►
      • Declare activity in manifest
      • Use <activity> tag under <application>
      • Specify android:name
    • Declaring Permissions
      ►
      • Use <uses-permission> tag
      • Must match app permissions
      • Examples: CALL_PHONE, INTERNET, READ_CONTACTS
