• Fragment
    ►
    • Introduction
      ►
      • Sub-activities with own views
      • Reusable across multiple activities
      • Added in Android 3.0 (API 11)
      • Improves performance and UX
      • Responsive to device type/orientation
    • Layouts
      ►
      • Single-pane: one panel shown
      • Multi-pane: multiple panels shown
      • Depends on screen space
    • Fragment Lifecycle
      ►
      • onAttach
      • onCreate
      • onCreateView
      • onActivityCreated
      • onStart
      • onResume
      • onPause
      • onStop
      • onDestroyView
      • onDestroy
      • onDetach
    • Fragment in Activity
      ►
      • Fragment lifecycle depends on Activity
      • Transactions: add, remove, replace
      • Back stack allows reverse navigation
    • Usage Steps
      ►
      • 1. Create layout XML (fragment_sample.xml)
      • 2. Create Fragment class (SampleFragment.java)
      • 3. Add fragment to activity (XML or code)
    • Fragment Manager & Transaction
      ►
      • FragmentManager: interacts with fragments
      • FragmentTransaction: atomic operations
      • Use addToBackStack for navigation support

      ►
      • DialogFragment
