• Windows Server Networking
    ►
    • Basic Network Concepts
      ►
      • Network: Linked NOS for sharing resources
      • NOS: Enables sharing of data, voice, video
    • Peer-to-Peer Network Model
      ►
      • All computers are equal (no server)
      • Ideal for small offices
      • Advantages
        ►
        • File/folder/printer sharing
        • Simple setup
        • Supports workgroups
    • Server-Based Network Model
      ►
      • Managed through one or more servers
      • Clients access server resources
      • Advantages
        ►
        • Multi-user access
        • Centralized management
        • Robust security
        • High performance
    • Using Windows Server
      ►
      • File/folder/printer sharing
      • Centralized or decentralized access
      • Software app management
      • Support for email, web, e-commerce
      • Central backups and upgrades
      • Time and cost efficient administration
    • Example Windows Server Services
      ►
      • Network management
      • Web-based management
      • Security and access control
      • Storage management
      • Remote access
      • Application services
      • Printer management
    • Server Clustering
      ►
      • Links multiple servers to act as one
      • Shares load and resources
      • Improves redundancy and performance
