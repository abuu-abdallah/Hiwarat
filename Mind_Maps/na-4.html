• Administrative Shares
    ►
    • What is an Administrative Share?
      ►
      • Automatically created by Windows
      • Ends with a dollar sign ($)
      • Used by admins, services, and tools
      • Not shown in net view or network browsing
    • Default Hidden Shares
      ►
      • C$, D$ – Root drive shares
      • ADMIN$ – %SYSTEMROOT% share
      • FAX$ – Used by fax clients
      • IPC$ – For remote admin via named pipes
      • PRINT$ – For remote printer admin
    • Creation & Deletion
      ►
      • Create Hidden Share
        ►
        • Use Computer Management → Shared Folders → New Share
        • Name must end in "$"
        • Set permissions (e.g., admins only)
      • Delete Hidden Share
        ►
        • Use Computer Management → Shares → Stop Sharing
        • System shares reappear after reboot
        • User-created hidden shares do not
    • Clarifications
      ►
      • Not all $-suffixed shares are administrative
      • Only system-defined ones are true admin shares
      • Windows XP Home doesn’t create them
    • Accessing Admin Shares
      ►
      • Use UNC path: \\ComputerName\C$
      • ADMIN$: \\ComputerName\ADMIN$
      • Only accessible with proper permissions
