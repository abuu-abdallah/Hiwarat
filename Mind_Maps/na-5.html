• IP Addressing
    ►
    • Internet Protocol (IP)
      ►
      • Protocol for addressing and routing
      • Used by TCP/IP suite
      • Provides logical addressing
    • IPv4 Addressing
      ►
      • 32-bit logical address
      • Dotted decimal notation (e.g., 192.168.3.24)
      • Each octet = 8 bits (0–255)
      • Format: w.x.y.z
    • Special IPv4 Addresses
      ►
      • 0.0.0.0 – Unspecified
      • 127.0.0.1 – Loopback
      • 255.255.255.255 – Limited Broadcast
      • 198.44.19.255 – Directed Broadcast
      • 198.44.19.0 – Network Number Address
    • Address Classes
      ►
      • Class A: 0–127
      • Class B: 128–191
      • Class C: 192–223
      • Class D: 224–239 (Multicast)
      • Class E: 240–255 (Experimental)
    • Subnet Masks
      ►
      • Divides network and host parts
      • Class A: 255.0.0.0
      • Class B: 255.255.0.0
      • Class C: 255.255.255.0
      • Example: 153.92.100.10 / 255.255.0.0 → Net: 153.92
      • Change subnet → changes net/host split
    • Octet Format
      ►
      • Class A: N.H.H.H
      • Class B: N.N.H.H
      • Class C: N.N.N.H
    • Broadcasting Types
      ►
      • Unicast – One-to-one
      • Multicast – One-to-group
      • Anycast – One-to-nearest
      • Broadcast – One-to-all
    • IPv6 Overview
      ►
      • 128-bit address, hex format
      • Auto-configuring, no subnet mask needed
      • Solves IPv4 limitations
      • Format: 2001:1b20:302:442a:110:2fea:ac4:2b
    • IPv6 Address Structure
      ►
      • 48-bit Public Topology
      • 16-bit Site Topology
      • 64-bit Interface Identifier
